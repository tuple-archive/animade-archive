A fun new way to engage children in coding

We helped volunteer-run Code Club groove their way to helping more children learn to code.

Code Club is a nationwide network of after-school coding clubs for 9-11 year-olds, led by volunteers. They were looking for fun new ways to engage children in coding and spread awareness of their brand. Code Club already used robots in their branding, so with this in mind we set out to create an interactive 'robot dance' tool and supporting promo film.

The key objective from the outset was to engage Code Club's young audience with coding, without alienating them. By creating a simple slider mode and an advanced code mode, we kept the barrier of entry low while showing how much fun coding can be. Making the dances shareable gives kids a sense of ownership whilst increasing awareness of the Code Club, and traffic to the website.

Have a go yourself. Make those robots dance!
